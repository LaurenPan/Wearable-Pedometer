Multitasking

When set to ON, enables APP to work in the background. Default is OFF.

Sample Rate

Sets the sample rate in seconds. The value of this property will determine how often data updates are sent to the delegate. Default value is 500 ms

Wheel Circ

Specifies the bike wheel circumference, in meters. This value is used to determine speed and distance in the bike sensors. Default value is 2070 mm

Stale String

The string returned by data formatting methods when data for the sensor is considered "stale". Default value is “--“

Stale Timeout

The stale data timeout, in seconds. If the time since the latest data was received from the sensor exceeds this value, the data for that sensor will be considered "stale". Default value is 5 seconds.

Coasting Timeout

The bike coasting timeout, in seconds. If no data is received for this duration, the bike is considered to be coasting. Default value is 3 seconds.
