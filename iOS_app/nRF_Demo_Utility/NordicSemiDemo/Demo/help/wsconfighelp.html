WEIGHT SCALE CONFIGURATION INFO

Proximity

When set to ON, enables proximity pairing. Default is OFF.

Wildcard

When set to ON, enables iPhone to connect to unpaired devices (devices not in the Manage Connections list). Default is OFF.

isUserProfielSelected

Has the user profile been selected.

userProfileId

User profile ID number.

Signal Eff

Returns the signal efficiency for the connected sensor.

  • For ANT connections, this property represents the ratio of messages received from the sensor over the number of messages expected (range [0, 1.0]). This value should give a good indication of the reception quality for the sensor.
  • For BT Smart connections, this property represents the RSSI (in dBm). If the sensor is not currently connected, the return value is -1.

Dev ID

Returns the Device ID of the connected sensor.

Sensor Info

Information about the sensor connected.

Manage Connections

List of paired devices. To delete a paired device from the list swipe left on the right hand side of the screen and press Delete.
