CONTINUOUS GLUCOSE METER INFO

Proximity

When set to ON, enables proximity pairing. Default is OFF.

Wildcard

When set to ON, enables iPhone to connect to unpaired devices (devices not in the Manage Connections list). Default is OFF.

TID

Transmitter ID for the sensor. Set to same value as written on the sensor.

PK

Permission Key for the sensor. Set to same value as written on the sensor.

ALERTS

Turn alarm signals on the phone on/off

High Alert - Give alarm if clucose level is above high threshold level

Low Alert - Give alarm if clucose level is below low threshold level

Rise Alert - Give alarm if glucose level is increasing too rapidly

Fall Alert - Give alarm if glucose level is decreasing too rapidly

curr dev time

Returns the current time of the sensor.

elapsed time

Returns the elapsed time since the sensor was started in seconds.

timestamp

Returns the time of the last glucose measurement.

rate of chng

Returns the glucose rate of change value from the sensor.

0 - Rate is Unavailable

1 - Rapidly Increasing, 8 to 3 mg/dL per minute

2 - Increasing, 3 to 2 mg/dL per minute

3 - Slowly Increasing, 2 to 1 mg/dL per minute

4 - Steady, 1 to -1 mg/dL per minute

5 - Slowly Decreasing, -1 to -2 mg/dL per minute

6 - Decreasing, -2 to -3 mg/dL per minute

7 - Rapidly Decreasing, -3 to -8 mg/dL per minute

alert high

Returns the status of the high alert from the sensor.

alert low

Returns the status of the low alert rom the sensor.

alert rising

Returns the status of the rising alert from the sensor.

alert falling

Returns the status of the falling alert from the sensor.

alert below 55

Returns the status of the "below 55" alert from the sensor.

Sensor Info

Information about the sensor connected.

Manage Connections

List of paired devices. To delete a paired device from the list swipe left on the right hand side of the screen and press Delete.
